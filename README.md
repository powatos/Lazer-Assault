# Lazer-Assault
A simple, terminal-based lazer obstacle game created using Potato Engine

This game was developed on Ubuntu 20.04.6 LTS  
View installation instructions under the release notes...

## Objective
Live as long as you can and get a high score! Lazers activate periodically and can damage you. Be careful, because you only have 100 health!

## Controls
Use `WASD` to move player in cardinal directions  
Press `Escape` to quit the game at any time
